```python
def label_func(f):
    if f[0].isupper():
        return 'cat' 
    else: 
        return 'dog' 
```


```python

```


```python
dls=ImageDataLoaders.from_name_func(path,files,label_func,item_tfms=Resize(244)) #데이터들을 분류 분석할 수 있는 함수.
```
